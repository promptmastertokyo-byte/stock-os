# 四季報・IR分析チェックリスト（95点版）

## STEP 0 会社概要・投資仮説

- [ ] 証券コード・市場・時価総額
- [ ] 主力事業と収益源
- [ ] なぜ今調べるのか
- [ ] 成長テーマ・業界順位
- [ ] 想定保有期間

## STEP 1 チャート（入口）

- [ ] 中長期トレンド
- [ ] 高値・安値・移動平均線
- [ ] 出来高の変化
- [ ] 決算前後の値動き

> チャートだけで買わず、調査優先度を決めるために使う。

## STEP 2 見出し・材料

- [ ] 最高益・増収増益
- [ ] 上方修正・増配・自社株買い
- [ ] 新製品・大型受注・M&A
- [ ] 海外展開・新市場参入
- [ ] 一時要因ではないか

## STEP 3 業績

- [ ] 売上高が中期で成長
- [ ] 営業利益が中期で成長
- [ ] EPSが中期で成長
- [ ] 営業利益率が改善
- [ ] 会社計画と市場予想の差
- [ ] 二期先も成長余地がある

## STEP 4 財務

- [ ] 自己資本比率（業種適正を考慮）
- [ ] ネットキャッシュ／ネットデット
- [ ] 営業CFが安定してプラス
- [ ] FCFが安定
- [ ] 有利子負債の返済余力
- [ ] 増資・転換社債・希薄化履歴

## STEP 5 収益性

- [ ] ROEが同業平均以上（目安10%以上、理想15%以上。金融・REITは業態基準）
- [ ] ROAの推移
- [ ] 営業利益率が同業より高い
- [ ] 投下資本利益率を説明できる
- [ ] 一時的な高収益ではない

## STEP 6 経営者・株主

- [ ] 経営者の実績と資本配分
- [ ] 創業者・オーナー持分
- [ ] 大株主の変化
- [ ] 社長交代の背景
- [ ] ストックオプション・希薄化
- [ ] ガバナンス上の懸念

## STEP 7 事業・競争優位性

- [ ] 成長市場にいる
- [ ] 継続収益・リピート性
- [ ] ブランド・技術・特許
- [ ] スイッチングコスト
- [ ] ネットワーク効果
- [ ] 参入障壁
- [ ] 顧客が選ぶ理由を説明できる

## STEP 8 同業比較

- [ ] 売上成長率
- [ ] 営業利益率
- [ ] ROE・ROA
- [ ] PER・PBR・PSR
- [ ] EV/EBITDA
- [ ] 時価総額
- [ ] 市場シェア

## STEP 9 バリュエーション

- [ ] PER
- [ ] PBR
- [ ] PSR
- [ ] PEG
- [ ] EV/EBITDA
- [ ] 過去レンジとの比較
- [ ] 同業との比較
- [ ] 強気前提がどこまで織り込まれているか

## STEP 10 株主還元

- [ ] 配当性向
- [ ] 配当の持続可能性
- [ ] 自社株買い
- [ ] 総還元性向
- [ ] 成長投資とのバランス

## STEP 11 リスク

- [ ] 借入・資金繰り
- [ ] 顧客集中・一社依存
- [ ] 為替・金利・原材料
- [ ] 景気循環
- [ ] 規制・地政学
- [ ] 技術代替
- [ ] 不祥事・ガバナンス
- [ ] 株価下落時に投資仮説が維持できるか

## STEP 12 判断と検証

- [ ] 買う理由を3つ以内で説明
- [ ] 見送る理由を3つ以内で説明
- [ ] 仮説が崩れる条件
- [ ] 売却・縮小条件
- [ ] 次回決算日
- [ ] 半年後レビュー日
- [ ] 1年後レビュー日

## 最終判定

> 配点は `rules/scoring-rules.md` を正とする。

- 企業品質（業績・財務・収益性・MOAT・経営）：`__/70`

- リスク耐性：`__/10`
- タイミング・需給：`__/5`
- 総合：`__/100`
- 確信度：低 / 中 / 高
