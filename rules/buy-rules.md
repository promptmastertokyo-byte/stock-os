# 買い判断ルール

## 基本方針

Stock OSのスコアは、買いを自動決定するものではない。企業品質・価格・タイミング・ポートフォリオ全体を分けて判断する。

## 深掘り条件

- 総合90点以上、または前回から5点以上改善
- 企業品質スコアが56/70以上
- 出典確認が完了している
- 反証条件を3つ以内で説明できる
- 同業比較が完了している

## 買い候補条件

- [ ] 投資仮説を3行で説明できる
- [ ] 企業品質が高い
- [ ] 現在価格に安全域がある
- [ ] 強気・標準・弱気シナリオを作成済み
- [ ] 下落時に追加調査する条件が明確
- [ ] 次回決算までの主要イベントを把握
- [ ] 一銘柄への過度な集中にならない

## 禁止事項

- 点数だけを理由に買わない
- SNSの話題だけで買わない
- 出典未確認のAI回答で買わない
- 決算直前に根拠なく飛び乗らない
- ナンピンを自動化しない
- 生活資金・緊急資金を使わない

## 記録

実際の売買情報は公開リポジトリに保存しない。公開版には投資仮説と検証可能な判断ルールのみ残す。
